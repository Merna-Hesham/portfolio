## Overview

Foodiee is a mobile application that allows users to explore a vast database of food recipes. It was developed as an ITI graduation project.

## Key Features

- **Recipe Discovery**: Search and filter recipes by ingredients, cuisine, or dietary restrictions.
- **API Integration**: Connects to third-party food APIs to fetch real-time recipe data.
- **User Friendly UI**: Clean and intuitive interface for seamless navigation.

## Technologies Used

- **Mobile**: Android (Native)
- **Languages**: Kotlin/Java
- **Backend**: REST APIs

## Outcome

Successfully delivered a functional MVP that demonstrates proficiency in native Android development and API consumption.
